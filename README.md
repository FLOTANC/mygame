# 🎮 FAMILY GAMES

子ども向けのインタラクティブゲーム集です。

## ゲーム一覧

| ファイル | ゲーム名 | 内容 |
|---|---|---|
| `firework.html` | Monster Sparkler | 指でなぞって花火を描くゲーム |
| `bird.html` | Lumibird | 横スクロール型アクションゲーム |
| `protein.html` | たんぱくしつどっち？ | タンパク質クイズゲーム |
| `color-mix.html` | いろをまぜよう | 色を混ぜる知育ゲーム |
| `game.html` | Earth Builder | 地球を作るゲーム |
| `flower.html` | Flower | 花のゲーム |

## 技術スタック

- HTML / CSS / JavaScript (バニラ)
- Three.js (firework.html)
- Web Audio API
- Canvas 2D API

## 起動方法

ローカルサーバーで開く（例: VS Code Live Server）か、そのままブラウザにドラッグ＆ドロップ。

```
index.html → ゲーム一覧ページ
```

## デプロイ

Vercel / Netlify などの静的ホスティングに対応しています。
